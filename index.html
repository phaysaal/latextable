LaTeX Table Editor

LaTeX Code

Ctrl+drag or Shift+click to select Auto-saved
Tips & Shortcuts

Navigation

Tab / Shift+Tab — Next / previous cell
Enter / Shift+Enter — Move down / up
↑ ↓ ← → — Arrow key navigation
Ctrl+A — Select all cells

Selection & Editing

Ctrl+Click + drag — Select cell range
Shift+Click — Extend selection to cell
Ctrl+C / X / V — Copy / Cut / Paste selected cells
Delete — Clear selected cells
Ctrl+Z / Ctrl+Shift+Z — Undo / Redo

Formatting

Ctrl+B — Bold  (\textbf{})
Ctrl+I — Italic  (\textit{})
Ctrl+U — Underline  (\underline{})
Toolbar: S strikethrough, TT monospace, Sc small caps
Color pickers for text color & cell background

Formulas (type = in a cell)

=sum(A1:A10) — Sum of range
=avg(A1:A10) — Average
=min() =max() =count() =median() =stdev()
=A1+B1*2 — Arithmetic with cell refs
=2*sum(A1:A5)+10 — Mix functions & math
=sqrt(A1) =abs() =round() =log() =sin() =A1^2
Cell refs: A1 = col A, row 1. Ranges: A1:C3

Context Menu

Right-click row numbers — Insert/delete rows
Right-click column headers — Insert/delete columns
Right-click any cell — Full context menu

Other Features

Drag column/row borders to resize
Per-row \hline checkboxes (left of row numbers)
Column types: l c r p{} m{} b{} with inner alignment
Paste from Excel/Sheets (tab-separated) auto-detected
CSV import/export via buttons in left panel
Auto-saved to browser — survives page reload
